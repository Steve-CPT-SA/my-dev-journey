Solar System.

    A pure HTML & CSS animated solar system — a creative rework of the FreeCodeCamp Ferris Wheel project.

About

    The sun is anchored to the left edge of the screen, with all 9 planets orbiting outward using CSS animations. Includes an asteroid belt between Mars and Jupiter, Saturn and Uranus rings, and twinkling background stars. Hover over any planet or the sun to reveal its name.

Tech Used

    -HTML5
    -CSS custom properties (:root variables, @property)
    -@keyframes animations
    -transform: rotate() and translateX() for orbital mechanics
    -radial-gradient, linear-gradient, and repeating-linear-gradient for planet and sun textures
    -border-radius and box-shadow for planet shapes and glow effects
    -data-name attribute with ::after pseudo-element for hover tooltips
    -pointer-events: none for interaction control

Notes
    Not scientifically accurate — orbit sizes and planet sizes are approximate for visual effect. Animation speeds are scaled for visual appeal rather than real orbital periods.
