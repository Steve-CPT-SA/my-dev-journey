04 — Paradoxical Motivation

    A cinematic, mobile-first motivational    web page built with pure HTML and CSS. No JavaScript. No frameworks.

About

    Five paradoxes of growth, each paired with AI-generated imagery. Dark aesthetic, handwritten typography, and a layout that shifts between mobile and desktop using CSS Grid.

Built With

    -HTML5 & CSS3
    -Google Fonts — Caveat
    -AI-generated images via DALL·E


Key Techniques

    CSS Grid with grid-template-areas — named layout areas that reflow between breakpoints
    Mobile-first typography — small by default, scaled up at min-width: 768px
    CSS Custom Properties for the design system
    object-fit: cover for consistent image dimensions


What I Learned

    How to correctly apply mobile-first thinking — setting defaults for mobile and scaling up, not the other way around. Also a much deeper understanding of grid-template-areas and how HTML structure affects grid behaviour.
