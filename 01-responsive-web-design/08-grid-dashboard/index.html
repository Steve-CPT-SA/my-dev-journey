02-solar-system

About this project

A pure HTML & CSS animated solar system.

The sun is anchored to the left edge of the screen, with all 9 planets orbiting outward using CSS animations.

Includes an asteroid belt between Mars and Jupiter, Saturn and Uranus rings, and twinkling background stars.

Technologies Used
  1. HTML5
  2. CSS3
    • CSS custom properties (:root variables, @property).
    • @keyframes animations.
    • transform: rotate() and translateX() for orbital mechanics.
    • radial-gradient, linear-gradient, and repeating-linear-gradient for planet and sun textures.
    • border-radius and box-shadow for planet shapes and glow effects.
    • data-name attribute with ::after pseudo-element for hover tooltips.
    • pointer-events: none for interaction control.

03-crawling-gecko

About this project

A top-down gecko illustration built entirely with CSS, featuring a full body, four articulated legs with forearms and feet, and a diagonal gait crawling animation.

Technologies Used
  1. HTML5
  2. CSS3
    • border-radius to create organic body shapes including the body, head, tail, and limb segments.
    • box-shadow to generate multiple toe pads and knuckle joints from single elements — avoiding the need for dozens of extra HTML elements.
    • transform: rotate() with transform-origin to position and animate each limb segment from its joint pivot point.
    • @keyframes with percentage stops to create a pause in the leg swing cycle.
    • animation-delay to offset diagonal leg pairs for a realistic gait.
    • steps() timing function for the body crawl movement.

07-flexbox-photo-gallery

About this project

Five Harmonies is a spiritual gallery exploring five Eastern philosophical traditions — Taoism, Shinto, Neo-Confucianism, Buddhism, and Zen.

Each tradition is represented by two images showing a seed-to-bloom progression: the first image introduces a core concept, the second builds on it.

The ten gallery images were generated using AI image generation tools (Microsoft Copilot) by crafting detailed prompts specifying style, palette, composition, and aspect ratio.

The visual style draws from Samurai Jack's bold ink-line structure combined with Studio Ghibli's atmospheric watercolour washes, with each tradition carrying its own dominant colour.

Technologies Used
  1. HTML5 (semantic elements: section, figure, figcaption).
  2. CSS3
    • Flexbox
    • Media queries
    • CSS reset

AWS AI & ML Challenge

About this project

About the Course I took part in the AWS AI & ML Challenge, where I explored the core concepts of artificial intelligence and machine learning, including generative AI, responsible AI, and practical ML techniques.

Project: Adaptive Productivity Coach As part of the challenge, I used prompt engineering on AWS PartyRock to build an AI-powered productivity app. What It Does The app takes in four user inputs and generates a structured, personalised plan:

1.Task — what the user wants to accomplish 2.Time — how much time is available 3.Energy — current energy level (1-10) 4.Focus Type — Deep Work, Learning, Light Tasks, Creative, Administrative, or Physical.

The output includes an adaptive step-by-step plan, a focus mode, a minimum win, and what to avoid — all tailored to the user's current state.

Why I Built It This app came from my own experience — working full time as an electrician while studying and coding after hours to transition into tech. Managing time and deciding what to do next without burning out is itself exhausting. I built this to solve that problem.
