Barad-Dûr — Pure CSS Art
A scene from Tolkien's Mordor, built entirely with HTML and CSS. No JavaScript, no images, no libraries — just markup and stylesheets.
This project is part of my personal project ladder, a creative rework of the FreeCodeCamp City Skyline exercise, reimagined as the Dark Tower of Barad-Dûr with Mount Doom erupting in the background.

What's in the scene

Barad-Dûr tower with the Eye of Sauron
Mount Doom with lava streams, fire and tephra
Smoke plumes rising from the volcano
Rocky terrain and ground layer
Background mountain range
A fiery Mordor sky


Technologies Used

HTML5
CSS3 — clip-path, polygon(), radial-gradient, linear-gradient, CSS custom properties, border-radius, viewport units (vw, vh)


Biggest Challenges
Getting to grips with clip-path: polygon() was the steepest part of this project in the beginning — understanding how percentage coordinates map to each element's own bounding box rather than the viewport took real experimentation. Accurate positioning across the scene and migrating everything from px to viewport units (vw/vh) to make the layout responsive added another layer of complexity.

What I Learned
That you can build surprisingly complex, atmospheric scenes using nothing but pure HTML and CSS. This project pushed my understanding of stacking contexts, coordinate systems, and how to think spatially when building with code.
