Responsive Navigation Bar
    -A simple responsive navigation bar built with HTML and CSS.
    -The navigation automatically adapts between desktop and mobile layouts using CSS media queries.

Desktop
    -Sticky navigation bar at the top
    -Horizontal layout using Flexbox

Mobile
    -Fixed navigation bar at the bottom
    -Responsive layout optimized for smaller screens

Technologies Used
    -HTML5
    -CSS3

CSS Concepts Practiced
    -@media
    -:root
    -position
    -z-index
    -flex-direction
    -justify-content
    -align-items
    -top
    -bottom
    -Flexbox

What I Learned
    -How to create responsive layouts using media query breakpoints
    -How to switch layouts between desktop and mobile views
    -How to use Flexbox for navigation alignment
    -How position: sticky and position: fixed behave
    -How to improve mobile usability with bottom navigation

Features
    -Responsive design
    -Sticky desktop navigation
    -Fixed mobile bottom navigation
    -Clean Flexbox layout
