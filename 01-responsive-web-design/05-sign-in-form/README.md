05 — Sign-in Form

    A reimagined sign-in form project, originally inspired by the Coddy.tech Netflix clone exercise. Built with semantic HTML and CSS only, no JavaScript.

Features

    -Floating label technique using CSS :focus, :valid, and the ~ sibling combinator
    -Glassmorphism card effect using backdrop-filter: blur()
    -CSS custom properties for consistent theming
    -Responsive layout using width: 100% and max-width
    -Accessible form structure using fieldset, legend, and paired label/input elements
    -Focus state styling for keyboard navigation
    -Hover transitions on the form card and button

Skills practised

    -HTML forms, semantic HTML, CSS positioning, Flexbox, CSS custom properties, transitions, responsive design
